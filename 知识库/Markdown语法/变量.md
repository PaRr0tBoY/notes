### 9.1 [[网页链接变量]]

- **网页链接变量 的格式：**
    1. 首先输入
        - **`[显示文本内容]`** + **`[变量名]`**
            - 变量名可以自己取，没啥限制，任意字符都可以
    2. 在文档任意一个区域，输入：
        - **`[变量名]`** + **:** + **空格** + 链接地址 （这个 **空格** 不打也没事)

```
[百度一下，你就知道][度娘]
[知乎-有问题，就会有答案][知乎]

<!-- 这里是变量区域 -->
[度娘]: http://www.baidu.com 
[知乎]: https://www.zhihu.com    
```

#### 示范

[百度一下，你就知道 712](http://www.baidu.com/)

[知乎-有问题，就会有答案 51](https://www.zhihu.com/)
